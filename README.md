# SQL Exercises Project

This repository contains solutions to SQL exercises based on flight, passenger, and payment data.

## Structure

- `1_flights_from_vladivostok.sql` – Companies operating flights from Vladivostok
- `2_passengers_to_moscow.sql` – Names of passengers flying to Moscow
- `3_flights_paris.sql` – Flight times from Paris to other cities
- `4_rustov_to_moscow_count.sql` – Count of flights from Rostov to Moscow
- `5_tu134_passengers.sql` – Passengers flying to Moscow on TU-134
- `6_flight_time_calculation.sql` – Calculating flight times (HH:MM:SS)
- `7_flights_between_10_and_14.sql` – Flights on Jan 1, 1900, from 10:00 to 14:59
